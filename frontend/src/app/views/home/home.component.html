Web application built with ANGULAR, a framework running over NODE.JS, using TYPESCRIPT, HTML and CSS. In addition, used FONT AWESOME for some icons. Hosted in a Linux UBUNTU dedicated server at DIGITALOCEAN cloud infraestructure. Version control by GIT and GITHUB. Backend running over JSON SERVER. Package management with NPM. Reverse proxy with NGINX, SSL certification with CERTBOT and Linux process management with PM2.
